# Privacy Policy

*Last updated: 2026-06-15*

## Summary

ewf-forensic is a local Rust library. It does not collect, transmit, or store any personal data on remote servers.

## Data Access

ewf-forensic reads only the file bytes you pass to it. All processing happens in memory on your local machine. Nothing is uploaded anywhere.

## Telemetry

ewf-forensic has **no telemetry**. It makes no network requests of any kind.

## Open Source

ewf-forensic is open source. You can audit every line of code at [github.com/SecurityRonin/ewf-forensic](https://github.com/SecurityRonin/ewf-forensic).
